# 📄 TEXO Document Master (Standardizer)

Ứng dụng độc lập giúp chuẩn hóa văn bản Word theo Nghị định 30/2020/NĐ-CP và Quy định chuyên môn nội bộ của TEXO.

## ✨ Tính năng chính
- **Chuẩn Nghị định 30:** Tự động chỉnh lề, font chữ (Times New Roman), cỡ chữ (14), giãn dòng (1.15) và sửa lỗi chính tả cơ bản.
- **Chuẩn Nội bộ TEXO:** Áp dụng 12 quy tắc vàng, giãn dòng Exactly 17pt, tự động in đậm "Kính gửi", xử lý lề Letterhead.
- **AI Spell Check:** Tự động rà soát và sửa các lỗi gõ nhanh phổ biến (quy ttrình -> quy trình, ko -> không...).

## 🚀 Cài đặt nhanh
1. Cài đặt thư viện: `pip install -r requirements.txt`
2. Chạy ứng dụng: `streamlit run app.py`
